欢迎来到 JBox API


Base url:
https://jbox.jiguang.cn
1、发送消息 (自定义集成)
message 向 webhook url （在集成编辑页面中获取） 发送 JSON payloads。
POST  v1/message/{integration_id}/{token}
Request Header:
    POST /v1/message/{integration_id}/{token}
        Accept: application/json

Request Body:
{
    "title": "今日头条",
    "message": "英国脱欧",
    "url": "https://jbox.jiguang.cn"
}
Request Parameters:
• title：消息的标题，可以为空。
• message：消息内容，不能为空。
• url：跳转 url ,手机端点击消息跳转的链接，为空不跳转(注意这个地方要加 http 前缀)。
Example Response:
Response Header
HTTP/1.1 200 OK    
Content-Type: application/json; charset=utf-8 
Response Data:
• None
